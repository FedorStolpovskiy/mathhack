Квадратные уравнение
Квадратные уравнения вычисляются
с помощью дискриминанта
a * x ^ 2 + b * x + c = 0
D = b ^ 2 - 4 * a * c,
x1 = (-b + d ^ (1/2)) / 2, x2 = (-b - d ^ (1/2)) / 2
где:
a, b, c - коэффициенты уравнения
x1, x2 - корни (искомые значения)
